# 语言功能说明

## 更新内容

已为 WallHaven 批量下载器添加了语言设置功能，支持英文和中文两种语言。

## 功能特性

### 1. 支持的语言
- **英文 (English)** - 默认语言
- **简体中文 (Chinese Simplified)**

### 2. 配置文件更新
在 [`wallhaven_config.json`](wallhaven_config.json:13) 中添加了 `language` 字段：
```json
{
  "language": "en"  // 默认为英文，可选值: "en" 或 "zh"
}
```

### 3. 设置界面更新
在设置窗口中添加了语言选择选项：
- 打开程序后，点击右上角的 "⚙️ 设置" 按钮
- 在设置窗口中可以看到 "语言" 选项
- 提供两个选项：
  - English (英文)
  - 简体中文 (中文)

### 4. 界面文本国际化
所有界面文本都已使用语言字典，包括：
- 窗口标题和按钮文本
- 输入框标签和占位符
- 错误和警告消息
- 下载日志和预览状态
- 设置窗口内容

## 使用方法

### 切换语言
1. 启动程序
2. 点击右上角的 "⚙️ 设置" 按钮
3. 在 "语言" 部分选择想要的语言
4. 点击 "应用" 按钮
5. 重启程序以完全应用新语言

### 注意事项
- 语言设置会保存到配置文件中，下次启动时自动加载
- 由于 CustomTkinter 的限制，已创建的控件文本无法动态更新
- 切换语言后需要重启程序才能完全应用新语言
- 程序会在切换语言时显示提示信息

## 技术实现

### 语言字典结构
在 [`wallhaven_gui_fixed.py`](wallhaven_gui_fixed.py:21) 中定义了 `LANGUAGES` 字典：
```python
LANGUAGES = {
    'en': {
        'title': 'Wallhaven Batch Downloader',
        'settings': 'Settings',
        # ... 更多文本
    },
    'zh': {
        'title': 'Wallhaven 壁纸下载器',
        'settings': '⚙️ 设置',
        # ... 更多文本
    }
}
```

### 语言加载
程序启动时从配置文件加载语言设置：
```python
self.current_language = config.get('language', 'en')  # 默认英文
self.lang = LANGUAGES.get(self.current_language, LANGUAGES['en'])
```

### 应用设置
在 [`apply_settings()`](wallhaven_gui_fixed.py:789) 方法中处理语言切换：
- 保存语言设置到配置文件
- 更新当前语言变量
- 更新语言字典引用
- 显示切换成功提示

## 默认设置

- **默认语言**: 英文 (en)
- **配置位置**: `wallhaven_config.json`
- **可选值**: "en" (英文) 或 "zh" (中文)

## 文件修改列表

1. [`wallhaven_config.json`](wallhaven_config.json:13) - 添加了 `language` 字段
2. [`wallhaven_gui_fixed.py`](wallhaven_gui_fixed.py:1) - 主要修改：
   - 更新了语言字典
   - 添加了语言选择界面
   - 实现了 `apply_settings()` 方法
   - 实现了 `update_ui_language()` 方法
   - 更新了所有界面文本使用语言字典
