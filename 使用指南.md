# Wallhaven 下载器 - 快速使用指南

## 🚀 快速开始

### 1. 运行程序
```bash
python wallhaven_gui.py
```

### 2. 基本操作流程

#### 步骤一：配置参数
1. **搜索关键词**：输入你想搜索的关键词（支持中文，会自动翻译）
   - 例如：`anime`、`风景`、`胡桃`、`landscape`
   
2. **下载数量**：设置要下载的图片数量
   - 例如：`100`
   
3. **保存位置**：选择图片保存的文件夹
   - 点击"浏览"按钮选择文件夹
   
4. **图片分类**（可多选）：
   - ☑ 常规 (General)
   - ☑ 动漫 (Anime)
   - ☐ 人物 (People)
   
5. **内容分级**（可多选）：
   - ☑ 安全 (SFW)
   - ☑ 擦边 (Sketchy)
   - ☐ 限制 (NSFW) - 需要API Key

#### 步骤二：开始下载
1. 点击 **"🚀 开始下载"** 按钮
2. 程序会先收集图片元数据（显示进度）
3. 然后自动开始批量下载原图
4. 日志窗口实时显示下载进度

#### 步骤三：预览图片
1. 元数据收集完成后，预览功能自动启用
2. 点击 **"⬅️ 上一张"** 或 **"下一张 ➡️"** 浏览图片
3. 预览区域显示：**"正在预览：第 X/总数 张"**
4. 预览使用缩略图，加载快速流畅

#### 步骤四：停止下载（可选）
- 如需中途停止，点击 **"⏸️ 停止下载"** 按钮
- 已下载的图片会保留

## 🔧 常见问题

### Q1: 预览按钮是灰色的？
**A**: 需要等待元数据收集完成，预览按钮会自动启用。

### Q2: 预览图片显示"加载中..."很久？
**A**: 可能是网络较慢，请耐心等待或检查网络连接。

### Q3: 下载速度慢？
**A**: 程序设置了防封机制，每张图片间隔1秒。这是正常的。

### Q4: 某些图片下载失败？
**A**: 个别图片失败不影响整体下载，程序会自动跳过并继续。

### Q5: 可以边下载边预览吗？
**A**: 可以！预览和下载完全独立，互不影响。

### Q6: 预览的是原图吗？
**A**: 预览使用的是缩略图（加载快），下载的是高清原图。
