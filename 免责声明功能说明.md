# 📜 免责声明功能集成说明

## 功能概述

已成功在 WallHaven Batch Downloader Pro 的设置界面中集成了"查看免责声明"功能。该功能通过一个醒目的按钮触发，弹出模态窗口展示双语免责声明内容。

---

## 🎯 实现的功能特性

### 1. **触发按钮**
- **位置**：设置窗口（[`open_settings()`](wallhaven_gui_fixed.py:1400)方法）的按钮区域顶部
- **样式**：红色醒目按钮（`#FF6B6B`），宽度 180px，高度 35px
- **文本**：
  - 中文：`📜 查看免责声明`
  - 英文：`📜 View Disclaimer`
- **优先级**：放置在"应用"和"关闭"按钮之上，确保用户优先看到

### 2. **弹窗设计**
- **窗口类型**：[`CTkToplevel`](wallhaven_gui_fixed.py:1289) - CustomTkinter 的置顶子窗口
- **窗口尺寸**：700x550 像素（适中大小，便于阅读）
- **窗口标题**：
  - 中文：`⚠️ 免责声明`
  - 英文：`⚠️ Disclaimer`
- **居中显示**：自动计算屏幕中心位置并定位窗口
- **置顶效果**：使用 `transient()` 方法保持窗口在主窗口之上

### 3. **文本展示**
- **组件**：[`CTkTextbox`](wallhaven_gui_fixed.py:1306) - 支持滚动的文本框
- **尺寸**：宽 650px，高 400px
- **字体**：Arial 12号
- **换行模式**：`wrap="word"` - 按单词智能换行
- **滚动支持**：自动支持鼠标滚轮滚动
- **只读状态**：`state="disabled"` - 禁止用户编辑内容

### 4. **关闭按钮**
- **位置**：弹窗底部居中
- **尺寸**：宽 120px，高 35px
- **功能**：点击后销毁弹窗（`destroy()`）

### 5. **模态效果**
- **实现方式**：使用 [`grab_set()`](wallhaven_gui_fixed.py:1297) 方法
- **效果**：用户必须先关闭免责声明窗口，才能继续操作主界面
- **优先级**：免责声明窗口优先级高于"年龄确认"窗口

### 2. 核心方法

#### [`show_disclaimer()`](wallhaven_gui_fixed.py:1282) 方法

```python
def show_disclaimer(self):
    """显示免责声明窗口"""
    # 1. 创建置顶子窗口
    disclaimer_window = ctk.CTkToplevel(self.root)
    disclaimer_window.title(self.lang['disclaimer_title'])
    disclaimer_window.geometry("700x550")
    
    # 2. 窗口居中显示
    disclaimer_window.update_idletasks()
    width = disclaimer_window.winfo_width()
    height = disclaimer_window.winfo_height()
    x = (disclaimer_window.winfo_screenwidth() // 2) - (width // 2)
    y = (disclaimer_window.winfo_screenheight() // 2) - (height // 2)
    disclaimer_window.geometry(f'{width}x{height}+{x}+{y}')
    
    # 3. 置顶并模态（锁定焦点）
    disclaimer_window.transient(self.root)
    disclaimer_window.grab_set()
    
    # 4. 添加标题标签
    title_label = ctk.CTkLabel(
        disclaimer_window,
        text=self.lang['disclaimer_title'],
        font=("Arial", 20, "bold")
    )
    title_label.pack(pady=(20, 10))
    
    # 5. 创建只读文本框
    textbox = ctk.CTkTextbox(
        disclaimer_window,
        width=650,
        height=400,
        font=("Arial", 12),
        wrap="word"
    )
    textbox.pack(padx=20, pady=10, fill="both", expand=True)
    textbox.insert("1.0", self.lang['disclaimer_content'])
    textbox.configure(state="disabled")  # 设置为只读
    
    # 6. 添加关闭按钮
    close_btn = ctk.CTkButton(
        disclaimer_window,
        text=self.lang['close'],
        width=120,
        height=35,
        font=("Arial", 14, "bold"),
        command=disclaimer_window.destroy
    )
    close_btn.pack(pady=(10, 20))
    
    # 7. 确保窗口获得焦点
    disclaimer_window.focus_set()
```

### 3. 设置窗口集成

在 [`open_settings()`](wallhaven_gui_fixed.py:1400) 方法的按钮区域添加：

```python
# 查看免责声明按钮（放在最上面，优先级最高）
disclaimer_btn = ctk.CTkButton(
    button_frame,
    text=self.lang['view_disclaimer'],
    width=180,
    height=35,
    font=("Arial", 13, "bold"),
    fg_color="#FF6B6B",      # 红色背景
    hover_color="#FF5252",   # 悬停时更深的红色
    command=self.show_disclaimer
)
disclaimer_btn.pack(pady=(5, 10))
```

## 🚀 使用方法

### 在主程序中使用

1. 启动 WallHaven Batch Downloader Pro
2. 点击主界面的 `⚙️ 设置` 按钮
3. 在设置窗口顶部找到红色的 `📜 查看免责声明` 按钮
4. 点击按钮查看完整的免责声明内容
5. 阅读完毕后点击 `关闭` 按钮
