## Description



**WallHaven Batch Downloader Pro** is a powerful desktop application that enables users to efficiently search, preview, and batch download high-quality wallpapers from Wallhaven.cc. Built with a modern dual-threaded architecture, it provides a seamless experience by allowing real-time image preview while downloading in the background.



### Key Features:



**🚀 Dual-Thread Architecture**

- Background thread handles automatic batch downloading of full-resolution images

- Main GUI thread remains responsive for smooth user interaction and preview navigation



**🖼️ Real-Time Preview System**

- Instant preview activation after metadata collection

- Browse images using large, easy-to-click navigation buttons (180x40px)

- Keyboard shortcuts support (Arrow keys: ←/→ or ↑/↓)

- Smart image caching for faster repeated viewing

- PIL-powered intelligent scaling maintains aspect ratios



**📊 Split-Panel Interface**

- Left panel: Real-time download progress logs

- Right panel: Interactive image preview with navigation controls

- Both panels operate independently without interference



**🔍 Advanced Search Capabilities**

- Automatic Chinese-to-English keyword translation

- Customizable category filters (General, Anime, People)

- Content rating selection (SFW, Sketchy, NSFW with API key)

- Configurable download quantity



**⚡ Performance Optimized**

- Asynchronous image loading prevents UI blocking

- Thumbnail-based preview for fast loading

- Full-resolution images downloaded in background

- Anti-ban protection with request throttling



**💾 User-Friendly Features**

- Configuration persistence across sessions

- Custom save directory selection

- Progress tracking with percentage display

- Pause/resume download capability



### Technical Highlights:

- Built with CustomTkinter for modern UI aesthetics

- PIL/Pillow for advanced image processing

- Thread-safe GUI updates using event queue

- Metadata pre-collection for efficient batch operations

- Callback-based architecture for real-time status updates



### Perfect For:

- Wallpaper enthusiasts seeking bulk downloads

- Digital artists collecting reference materials

- Content curators building image libraries

- Anyone who wants to browse before downloading



**WallHaven Batch Downloader Pro** transforms the tedious process of downloading multiple wallpapers into an efficient, enjoyable experience with its preview-while-downloading capability and intuitive interface.
